# 📘 Operadores aritméticos

| Operador | Significado      |
| -------- | ---------------- |
| +        | adição           |
| -        | subtração        |
| *        | multiplicação    |
| /        | divisão          |
| \        | divisão inteira  |
| % ou mod | resto da divisão |
| ^        | potenciação      |
# 📘 Operadores relacionais

|Operador|Significado|
|---|---|
|=|igual|
|<>|diferente|
|>|maior que|
|<|menor que|
|>=|maior ou igual|
|<=|menor ou igual|
