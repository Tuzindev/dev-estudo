#### Comando "leia"
Para fazer o programa ler um valor e armazena-lo em uma variável, usa-se  comando "leia":

leia(x)
Ele permite que o usuário escreva um valor para ser armazenado na variável 

```
Algoritmo "semnome"

Var
   idade: inteiro
   salario1, salario2: real
   nome: caractere

Inicio
   escreval("Qual o seu nome? ")
   leia(nome)
   escreval("Qual a sua idade? ")
   leia(idade)
   escreval("Digite quanto você ganhou nos dois útlimos meses: ")
   leia(salario1)
   leia(salario2)
   
   escreval("DADOS DIGITADOS:")
   escreval("Nome = ", nome)
   escreval("Idade = ", idade," anos")
   escreval("Salario 1 = ", salario1:8:2)
   escreval("Salario 2 = ", salario2:8:2)

Fimalgoritmo
```
