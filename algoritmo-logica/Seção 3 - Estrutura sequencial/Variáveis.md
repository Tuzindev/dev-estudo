##### Definição informal:
Em programação, uma variável é uma porção de memória (RAM) utilizada para armazenar dados durante a execução dos programas.
As variáveis são armazenadas na memora RAM
##### Declaração de variáveis 
**Sintaxe:**
```
<nome> : <tipo>
```
**Exemplos:**
```
idade : inteiro
altura : real
logadouro : caractere
```
Aqui basicamente estamos criando uma variável e dando um tipo pra ela.
No caso acima a variável idade recebe o tipo inteiro, sendo possível colocar um número nela.

**Características de uma variável:**
- Nome (ou identificador)
- Tipo
- Valor
- Endereço da memória
##### Tipos básicos em VisualG
- Inteiro = número inteiro
- real = número com ponto flutuante
- caractere = texto
- logico = valor verdade
##### Regras para nome de variáveis
- Não pode começar com dígito: use uma letra ou _
- Não pode ter espaço em branco
- Não pode usar acentos ou til
- Use o padrão "camelCase"
Errado:
```
5minutos : inteiro
salário : inteiro
salário do funcionario : inteiro
```
Correto:
```
_5minutos : inteiro
salario : inteiro
salarioDoFuncionario : inteiro
```
