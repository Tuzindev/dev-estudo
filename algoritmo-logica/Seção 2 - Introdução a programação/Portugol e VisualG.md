## O que é Portugol  
É uma forma didática de escrever algoritmos, parecida com português estruturado.  
  
## O que é VisualG  
É uma ferramenta usada para escrever, testar e executar algoritmos em Portugol.  
  
## Para que servem  
- aprender lógica de programação  
- praticar algoritmos  
- testar estruturas básicas  
  
## Vantagens  
- fácil para iniciantes  
- ajuda a entender lógica antes de ir para uma linguagem real
