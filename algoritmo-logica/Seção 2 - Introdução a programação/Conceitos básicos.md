## Código-fonte  
É o código escrito pelo programador em uma linguagem de programação.  
  
## Código-objeto  
É o código gerado após a compilação.  
  
## Compilador  
Programa responsável por traduzir o [[Código-fonte]] para [[Código-objeto]].  
  
## Interpretador  
Executa o [[Código-fonte]] diretamente, instrução por instrução.  
  
## Máquina Virtual  
Ambiente que executa um código intermediário, como bytecode.  
  
## Sistema Operacional  
Não entende diretamente o código-fonte, por isso é necessário um processo de tradução.
