## O que é
É o conjunto de regras léxicas e sintáticas que formam programas.

- Léxica = ortografia
- Sintática = gramática

## Léxica
Verifica se os elementos isolados estão escritos corretamente.

### Exemplo em português
- cachorro → correto
- caxorro → errado

### Exemplo em programação
- main → correto
- maim → errado

## Sintática
Verifica se a estrutura da instrução está correta.

### Exemplo em português
- O cachorro está com fome → correto
- A cachorro está com fome → errado

### Exemplo em programação
- x = 2 + y; → correto
- x = + 2 y; → errado

## Ideia principal
Uma linguagem de programação precisa ter palavras válidas e também uma estrutura válida.
